Quick Start

<!-- HTML: Just use data-bind -->
<div>
  <p>Count: <span data-bind="count"></span></p>
  <input data-bind="name" placeholder="Enter name">
  <button id="increment">+</button>
</div>

<script type="module">
  import { state, bindDom } from 'lume-js';

  // Create reactive state
  const store = state({ count: 0, name: 'World' });

  // Bind to DOM - that's it!
  bindDom(document.body, store);

  // Update state with standard JavaScript
  document.getElementById('increment')
    .addEventListener('click', () => store.count++);
</script>

Why Choose Lume.js?

🎯 Standards-Only

Uses only data-* attributes and standard JavaScript. Your HTML validates perfectly and screen readers work flawlessly.

📦 Tiny Bundle

Less than 2KB gzipped. No virtual DOM overhead, just direct DOM updates that are fast and efficient.

🔧 No Build Step

Works directly in the browser. Import from CDN or use with any bundler - your choice.

🎨 No Lock-in

Works with any library (GSAP, jQuery, D3, etc.). Mix and match with your existing stack.

⚡ Fast Performance

Direct DOM updates with surgical precision. Only what changed gets updated.

🧹 Clean API

Cleanup functions prevent memory leaks. Proper lifecycle management built-in.
