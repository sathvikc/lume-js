# Lume.js Examples

This directory contains interactive examples demonstrating Lume.js features.

## Running Locally

```bash
npm run dev
```

Then navigate to `http://localhost:5173/examples/`

## Building for Production

```bash
npm run build:examples
```

This generates a static build in the `dist/` folder that can be deployed to GitHub Pages or any static hosting service.

## Available Examples

- **todo**: Classic TodoMVC implementation
- **tic-tac-toe**: Interactive game demonstrating state management
- **form-heavy**: Complex form handling with validation
- **comprehensive**: Showcase of all Lume.js features

Each example demonstrates different aspects of Lume.js:
- Reactive state management
- DOM binding
- Computed values
- Watchers and effects
- Event handling
